DMT Screen Saver

DmtWallpaper.scr is a screen saver that will be copied to the Windows system directory when the msi installer is used. If you are doing a portable/zip install, then you will need to copy this file manually.

You will need to use Windows Screen Saver settings to activate this screen saver.

This screen saver works with the DMT Wallpaper Changer module to show the DMT wallpaper as your screen saver. This means that DMT must be running and the Wallpaper Changer module set to change the wallpaper for this to work.
