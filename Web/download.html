Requirements

Dual Monitor Tools will work with any version of Windows that has Microsoft .NET Framework version 2.0 or later installed.

If you do not have a suitable version of .NET installed, then assuming your version of Windows supports it, you can install it from Windows Update. You should be able to find it in the Optional Updates section.

Dual Monitor Tools has been tested and found to work on XP, Vista and Windows 7.

Download

Dual Monitor Tools is hosted by SourceForge.Net and the latest version may be downloaded from here.

Installation

There is currently no installer for Dual Monitor Tools, so you will need to unzip the file you downloaded into the directory you want to run it from.

If you have installed a previous version of Dual Monitor Tools, then you will need to make sure that none of these tools are currently running. Remember that SwapScreen, DualLauncher and DualSnap run from the notification area of the taskbar, so to close these tools down you will need to right click on the corresponding icon and select Exit.

Running

If when you run any of the tools, you get a message along the lines of The dynamic link library mscoree.dll could not be found... then this probably means you do not have .NET 2.0 installed.

See the tool specific pages to see how to use each tool.

Uninstall

To uninstall the program, you will need to remove any shortcuts you manually added, remove the files you unzipped and remove the configuration and data files for any of the tools you have used.

The configuration files for the tools may be found under %LOCALDATA%\GNE. Typically this directory is something like C:\Users\<UserName>\AppData\Local\GNE.

If you have nothing else in %LOCALAPPDATA%\GNE, then it should be safe to delete this directory as well.
