11 Jan 2014 - release 1.9

Swap Screen

  • New hotkey to swap the positions of the top two windows.
  • New keys for snapping windows to the left/right/up/down.
  • Improved handling of configuration saving problems.

Dual Wallpaper

  • Supports drag and drop. (Thanks to Nithin Philips for this.)
  • Supports DPI scaling other than 100%.
  • Supports .jpeg (with an e) extension for input images.
  • Support added for Windows 8 and 8.1.

13 Apr 2011 - release 1.8

Swap Screen

  • Can now move the active window to any of 10 user defined areas on your screens.
  • Option to allow sticky/locked cursors to return freely to the primary screen.
  • Fixed problem where cancelling the 'Change hotkey' dialog did not always leave the hotkey unchanged.

Dual Launcher

  • Fixed problem where disabling the hotkeys did not work.

Dual Wallpaper

  • The background colour can now be changed. (Thanks to Shawn Steele for this.)

DisMon

  • -n option added to not restore screens on exit.
  • -x option added to change screens and exit.

Dual Snap

  • Fixed problem where cancelling the 'Change hotkey' dialog did not always leave the hotkey unchanged.

23 Jan 2011 - release 1.7

Swap Screen

  • Cursor locking/stickiness problems with some screen layouts with different resolutions now fixed.
  • Fix to mouse/keyboard hooking.
  • Can now set default cursor mode at startup.
  • More keys now available to be used as hotkeys.

Dual Launcher

  • Fixed problem where the entry window could forget its default position.
  • More keys now available to be used as hotkeys.

Dual Wallpaper

  • No longer crashes if trying to save wallpaper to a file that is being used to compose the wallpaper.

DisMon

  • Unchanged.

Dual Snap

  • Now has a hotkey to toggle the display of the snap on the second monitor.
  • More keys now available to be used as hotkeys.
