2 Oct 2016 - release 2.4

DMT

  • New options panel to show monitor configuration.
  • Can change primary monitor using command line, magic word or GUI.
  • Can change monitor brightness using command line, magic word or GUI.
  • Existing instance of DMT can be closed from the command line.
  • Cursor fix to remember free movement key.
  • Snap fix for snaps that are smaller then secondary on one dimension only.
  • SwapScreen fix for 'Next Screen' when taskbar at left or top of monitor.
  • Wallpaper changer obeys exif orientation flag.
  • Wallpaper changer temporary fix for Unsplash images.

2 Feb 2016 - release 2.3

An installer is now available.

DMT

  • SwapScreen fix for when swapping top 2 windows.
  • WallpaperChanger makes sure correct wallpaper mode used if using smooth changes.
  • Cursor fix for cursor not sticky after Ctrl-Alt-Del.
  • No longer allows multiple instances of its GUI interface to be run.

19 Dec 2015 - release 2.2

DMT

  • Magic words can be run from a batch file with: DMT "DMT:Launcher:Activate magic-word".
  • Snap hotkey added to snap active window.
  • Snap snap shown same size (pixel basis) as original with options to scale.
  • WallpaperChanger LocalDisk can specify different directories for each monitor.
  • WallpaperChanger LocalDisk can specify a single image rather than a directory.

1 Nov 2015 - release 2.1

DMT

  • Actions can now be performed from the command line / batch files.
  • File locations are shown in options for troubleshooting.
  • Cursor Option to allow mouse buttons to be used to cross cursor boundaries.
  • Cursor Added hotkey to move cursor to center of primary monitor.
  • SwapScreen fixed hotkey to move active window to previous monitor.
  • WallpaperChanger has crossfade option for changing wallpaper.
  • WallpaperChanger shows details of images used in wallpaper.
  • WallpaperChanger Flickr added as an image provider.
  • WallpaperChanger Can choose different local disk folder for monitors in portrait orientation.
  • WallpaperChanger Unsplash provider updated for changes to their website.

17 Jun 2015 - release 2.0

New tool DMT provides the functionality previously provided by SwapScreen, DualLauncher, DualSnap and DualWallpaperChanger. Handling of configuration files has been changed to make it easier to use as a portable app. DisMon has been dropped for the time being.

11 Jan 2014 - release 1.9

Swap Screen
