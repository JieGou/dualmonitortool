Source for latest released version

The source for the latest released version of File Locater can be downloaded from the SourceForge download page

Subversion access

Alternatively you can access the code directly using subversion.

If you have a subversion client installed, then you can check out the very latest code with:

svn co https://dualmonitortool.svn.sourceforge.net/svnroot/dualmonitortool/trunk DualMonitorTools

To check out a particular version, say 1.0, then this can be checked out with:

svn co https://dualmonitortool.svn.sourceforge.net/svnroot/dualmonitortool/tags/rel-1.0 DualMonitorTools

Requirements

The following tools have been used to build Dual Monitor Tools:

  • Visual C# 2005 Express Edition

Directory layout

The following sub-directories exist in the main project directory:
(Note: the directory Web is only available under svn)

SwapScreen
Contains the C# source for Swap Screen.
DualSnap
Contains the C# source for Dual Snap.
Web
Contains this web site.

Note there is currently some duplication of code between SwapScreen and DualSnap. At some point in the future, either a common library will be used for this code, or possibly the two tools could be merged into one.

XML documentation

All significant public methods use XML documentation tags. You can use your favourite tool e.g. Doxygen to convert these to HTML viewable documentation.
